# SafePak Employee Attendance Management System

## Overview

SafePak Employee Attendance Management System is a full-stack web application developed to manage employee attendance records and authentication.

The system allows administrators to register employees, monitor attendance records, and manage employee access through a secure login system.

## Features

### Admin Features

* Secure Admin Login
* Add New Employees
* View Attendance Records
* Logout Functionality
* JWT Authentication

### Employee Features

* Secure Employee Login
* Attendance Tracking
* Role-Based Access

## Technologies Used

### Frontend

* React
* Axios
* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js
* MySQL
* JWT Authentication

### Deployment

* Netlify (Frontend)
* Railway (Backend)

## System Architecture

Frontend (React) → Backend API (Node.js/Express) → MySQL Database

## Installation

### Clone Repository

```bash
git clone https://github.com/muindikyalo1998/safepak-system.git
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Run Backend

```bash
npm start
```

### Run Frontend

```bash
npm start
```
